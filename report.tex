\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}
In this internship, I developed an Internet of Things (IoT) based Smart Health and Location Monitoring System. I broke the project down into three main phases so that testing the parts would be easier.

\textbf{Phase 1} focused on wiring up and testing the individual sensors: the MAX30102 for heart rate and SpO2, the MLX90614 for body temperature, and the NEO-6M GPS module for location tracking. 

In \textbf{Phase 2}, I connected these sensors to an ESP32 microcontroller, which pushed the sensor data over Wi-Fi using MQTT. I also built a web dashboard with React.js and Firebase and hosted it on Vercel so anyone could view the data live. 

In \textbf{Phase 3}, I upgraded the main brain of the system to a Raspberry Pi 5 to handle more complex Python scripts. I also moved the messy breadboard circuit onto a soldered Zero PCB (perfboard) to make it smaller and stronger, and I built a mobile app using React Native (Expo Go) so users could check the data right from their phones.

This report explains how I wired the hardware, the math I used for signal processing, how the software works, and my final results.

\chapter{Introduction}
\section{Background and Motivation}
Tracking patient health and location in real-time is very important for elderly care and for people working in dangerous environments. While there are commercial smartwatches out there, they are usually expensive and hide all their raw data behind closed software ecosystems. 

For this project, I decided to build a custom alternative from scratch. By using standard off-the-shelf components, I kept the cost low. More importantly, this setup gives me full access to the raw sensor data, meaning I can write my own signal processing code and own all my data on Firebase.

\section{Problem Statement}
Wiring up multiple I2C and UART sensors to one microcontroller is tricky. My goal was to read heart rate, body temperature, and location, crunch the numbers, and send it all to a web server. I also had to shrink the hardware from a bulky breadboard down to a small soldered PCB so someone could actually wear it, and I needed to build a proper app interface to show the data.

\section{Project Phasing}
To keep myself organized, I divided the work into three phases:
\begin{enumerate}
    \item \textbf{Phase 1:} Testing each sensor one by one and writing the basic Arduino code.
    \item \textbf{Phase 2:} Hooking the sensors up to an ESP32, getting MQTT working, and coding a Vercel web dashboard.
    \item \textbf{Phase 3:} Moving everything to a Raspberry Pi 5, soldering the final PCB, and building an Expo Go mobile app.
\end{enumerate}

#include <Wire.h>
#include "MAX30105.h"
#include "heartRate.h"

MAX30105 particleSensor;
const byte RATE_SIZE = 4;
byte rates[RATE_SIZE];
byte rateSpot = 0;
long lastBeat = 0;
float beatsPerMinute;
int beatAvg;

void setup() {
  Serial.begin(115200);
  if (!particleSensor.begin(Wire, I2C_SPEED_FAST)) {
    Serial.println("MAX30102 was not found. Please check wiring.");
    while (1);
  }
  particleSensor.setup();
  particleSensor.setPulseAmplitudeRed(0x0A);
}

void loop() {
  long irValue = particleSensor.getIR();
  if (checkForBeat(irValue) == true) {
    long delta = millis() - lastBeat;
    lastBeat = millis();
    beatsPerMinute = 60 / (delta / 1000.0);
    if (beatsPerMinute < 255 && beatsPerMinute > 20) {
      rates[rateSpot++] = (byte)beatsPerMinute;
      rateSpot %= RATE_SIZE;
      beatAvg = 0;
      for (byte x = 0 ; x < RATE_SIZE ; x++) beatAvg += rates[x];
      beatAvg /= RATE_SIZE;
    }
  }
  Serial.print("IR = ");
  Serial.print(irValue);
  Serial.print(", BPM = ");
  Serial.println(beatAvg);
}
\end{lstlisting}

#include <SoftwareSerial.h>
#include <TinyGPS++.h>

static const int RXPin = 4, TXPin = 3;
static const uint32_t GPSBaud = 9600;

TinyGPSPlus gps;
SoftwareSerial ss(RXPin, TXPin);

void setup() {
  Serial.begin(115200);
  ss.begin(GPSBaud);
}

void loop() {
  while (ss.available() > 0) {
    if (gps.encode(ss.read())) {
      if (gps.location.isValid()) {
        Serial.print("Latitude: ");
        Serial.print(gps.location.lat(), 6);
        Serial.print(", Longitude: ");
        Serial.println(gps.location.lng(), 6);
      } else {
        Serial.println("Waiting for valid GPS lock...");}}
  }
}

\chapter{Mathematical Signal Processing}
To actually get accurate heart rate and SpO2 readings from the MAX30102, I couldn't just read the raw optical data; I had to process it using some math in my code.

\section{Photoplethysmography and the Beer-Lambert Law}
Calculating blood oxygen (SpO2) depends on the fact that oxygenated hemoglobin ($HbO_2$) and deoxygenated hemoglobin ($Hb$) absorb light differently. This is based on the Beer-Lambert Law~\cite{webster1997design}, which states that the absorbance $A$ of light passing through a material is:
\begin{equation}
    A = \ln\left(\frac{I_{in}}{I_{out}}\right) = \varepsilon \cdot c \cdot d
\end{equation}
Here, $\varepsilon$ is how much light the blood absorbs, $c$ is the concentration of hemoglobin, and $d$ is the distance the light travels through the tissue.

\section{The Ratio of Ratios Algorithm}
Because I don't know the exact distance $d$ the light travels through the user's skin, my code separates the changing part of the pulse signal (the AC component) from the steady baseline (the DC component) for both the Red and Infrared (IR) LEDs.

This gives us the "Ratio of Ratios" ($R$):
\begin{equation}
    R = \frac{\left( \frac{AC_{red}}{DC_{red}} \right)}{\left( \frac{AC_{ir}}{DC_{ir}} \right)}
\end{equation}
Once my code calculates $R$, I use a standard linear formula to estimate the SpO2 percentage:
\begin{equation}
    SpO_2 (\%) = 110 - 25 \cdot R
\end{equation}
I run this calculation continuously on the microcontroller before it ever sends any data over the internet.

\chapter{Phase 3: Raspberry Pi 5 and Mobile App Integration}
\section{Migrating to Raspberry Pi 5}
To handle more data and run Python scripts, I swapped out the ESP32 for a Raspberry Pi 5~\cite{rpi5}. The Pi 5 is basically a full computer, which let me use Python's \texttt{asyncio} library. This was a game changer because it let me read sensors and push data to the internet simultaneously without the system freezing up.

\begin{lstlisting}[language=Python, caption=Simplified Python asyncio loop for non-blocking sensor polling on the Raspberry Pi 5.]
import asyncio

async def poll_sensors():
    while True:
        # Read MAX30102 and MLX90614 via I2C
        ir_val, red_val = max30102.read_fifo()
        temp = mlx90614.get_object_temp()
        
        process_dsp(ir_val, red_val)
        
        # Yield control back to the event loop
        await asyncio.sleep(0.01) 

async def publish_telemetry():
    while True:
        # Publish batch data to Firebase every 15 seconds
        if network_connected():
            send_to_firebase(get_latest_data())
        await asyncio.sleep(15.0)

async def main():
    # Run both tasks concurrently without multithreading
    await asyncio.gather(poll_sensors(), publish_telemetry())

if __name__ == "__main__":
    asyncio.run(main())
\end{lstlisting}

\section{Zero PCB Fabrication and Miniaturization}
The breadboard circuit was too bulky, so I soldered all the components onto a Zero PCB (perfboard). The final soldered board ended up being way smaller and a lot more stable.

% --- CHAPTER 7 ---
\chapter{System Challenges and Troubleshooting}
I ran into a few annoying hardware issues while building this. Here is how I solved them.

\section{Logic Level Voltage Mismatches}
\textbf{Challenge:} The Raspberry Pi 5 expects 3.3V logic, but the NEO-6M GPS works best when powered by 5V. If I plugged that 5V signal directly into the Pi's RX pin, it would have destroyed it. \\
\textbf{Solution:} I wired bidirectional logic level shifters (using BSS138 transistors) between the 5V sensors and the Pi. This safely dropped the 5V signals down to 3.3V right before they hit the Pi's pins.

\section{GPS Cold Start Latency}
\textbf{Challenge:} Whenever I went outside to test, the NEO-6M GPS took way too long (3 to 5 minutes) to find satellites and get a lock. \\
\textbf{Solution:} I soldered a coin-cell battery to the back of the GPS module. This kept the satellite data saved in its memory even when I unplugged the main power, allowing the GPS to do a "hot start" and find a lock in just a few seconds. I also added an if-statement in the code to ignore the GPS data if it hadn't found a lock yet.

\section{Motion Artifacts in PPG Sensing}
\textbf{Challenge:} The MAX30102 sensor is super sensitive to movement. If my finger moved around on it, the heart rate readings would jump to crazy high numbers. \\
\textbf{Solution:} I wrote a moving average filter in my code to smooth out the noisy data. I also added a hard limit to ignore any heart rate values that were physically impossible (like anything over 220 BPM).

log = logging.getLogger("T-BTN")

GPS_PORT = "/dev/serial0"
GPS_BAUD = 9600

PUBLISH_INTERVAL = 15.0
DEVICE_ID = "tbtn-001"
GPS_TIMEOUT = 0.01

POLL_INTERVAL  = 0.01
PRINT_INTERVAL = 1.0
MLX_INTERVAL   = 0.5
DSP_INTERVAL   = 1.0

SAMPLE_RATE_HZ = int(1 / POLL_INTERVAL)
WINDOW_SIZE    = SAMPLE_RATE_HZ * 4
MA_KERNEL      = int(SAMPLE_RATE_HZ * 1.5)

PEAK_MIN_DISTANCE     = int(SAMPLE_RATE_HZ * 0.40)
PEAK_THRESHOLD_FACTOR = 0.25
MIN_PULSE_AMPLITUDE   = 3.0

BPM_MAX_STEP    = 20
BPM_EMA_ALPHA   = 0.30

SPO2_A = 115.0
SPO2_B = 10.0

ADC_MAX = 262143
SATURATION_FRACTION_LIMIT = 0.3

NO_CONTACT_IR_THRESHOLD = 20000

DEFAULT_LED_CURRENT = max30100.LED_CURRENT_4_4MA

MLX90614_I2C_BUS_NUMBER = 3

def moving_average(signal: np.ndarray, kernel: int) -> np.ndarray:

    if kernel < 2:
        return signal.copy()
    kernel = min(kernel, len(signal))
    cumsum = np.cumsum(np.insert(signal, 0, 0))
    ma = (cumsum[kernel:] - cumsum[:-kernel]) / kernel
    pad = np.full(kernel - 1, signal[0] if len(signal) else 0.0)
    return np.concatenate([pad, ma])

def detect_peaks(signal: np.ndarray, min_distance: int, threshold_factor: float, min_amplitude: float = MIN_PULSE_AMPLITUDE):

    if len(signal) < 3:
        return np.array([], dtype=int)

    sig_mean = np.mean(signal)
    sig_range = np.max(signal) - np.min(signal)

    if sig_range < min_amplitude:
        return np.array([], dtype=int)

    threshold = sig_mean + threshold_factor * sig_range

    peaks = []
    last_peak = -min_distance

    for i in range(1, len(signal) - 1):
        if (signal[i] > signal[i - 1]
                and signal[i] > signal[i + 1]
                and signal[i] > threshold
                and (i - last_peak) >= min_distance):
            peaks.append(i)
            last_peak = i

    return np.array(peaks, dtype=int)

def compute_bpm_spo2(ir_buf: collections.deque,
                     red_buf: collections.deque,
                     sample_rate: float):

    if len(ir_buf) < WINDOW_SIZE:
        return None, None

    ir_arr  = np.array(ir_buf,  dtype=np.float64)
    red_arr = np.array(red_buf, dtype=np.float64)

    if (np.mean(ir_arr >= ADC_MAX - 1) > SATURATION_FRACTION_LIMIT
            or np.mean(red_arr >= ADC_MAX - 1) > SATURATION_FRACTION_LIMIT):
        return "SATURATED", None

    ir_dc  = moving_average(ir_arr,  MA_KERNEL)
    red_dc = moving_average(red_arr, MA_KERNEL)

    ir_ac  = ir_arr  - ir_dc
    red_ac = red_arr - red_dc

    lp_kernel = max(1, int(sample_rate * 0.08))
    ir_ac = moving_average(ir_ac, lp_kernel)
    red_ac = moving_average(red_ac, lp_kernel)

    bpm = None
    peaks = detect_peaks(ir_ac, PEAK_MIN_DISTANCE, PEAK_THRESHOLD_FACTOR)
    log.debug("IR AC range=%.1f peaks=%d", float(np.max(ir_ac) - np.min(ir_ac)), len(peaks))
    if len(peaks) >= 2:
        avg_dist = np.mean(np.diff(peaks))
        if avg_dist > 0:
            candidate = round((sample_rate / avg_dist) * 60.0)
            if 40 <= candidate <= 220:
                bpm = candidate

    spo2 = None
    if bpm is not None:
        try:
            dc_ir  = np.mean(ir_dc)
            dc_red = np.mean(red_dc)
            if dc_ir > 1 and dc_red > 1:
                ac_ir  = float(np.std(ir_ac))
                ac_red = float(np.std(red_ac))
                if ac_ir > 0 and ac_red > 0:
                    R   = (ac_red / dc_red) / (ac_ir / dc_ir)
                    val = SPO2_A - SPO2_B * R
                    if val > 100.0:
                        val = 100.0
                    if 70.0 <= val <= 100.0:
                        spo2 = round(val)
        except ZeroDivisionError:
            pass

    return bpm, spo2

class BPMSmoother:

    MEDIAN_WINDOW = 9

    def __init__(self, max_step=BPM_MAX_STEP, alpha=BPM_EMA_ALPHA):
        self.max_step = max_step
        self.alpha = alpha
        self.value = None

        self._history = collections.deque(maxlen=self.MEDIAN_WINDOW)

    def update(self, new_bpm):

        if new_bpm is None:
            return round(self.value) if self.value is not None else None

        self._history.append(float(new_bpm))

        median_bpm = sorted(self._history)[len(self._history) // 2]

        if self.value is None:
            self.value = median_bpm
        else:

            if abs(median_bpm - self.value) > self.max_step:
                return round(self.value)
            self.value = (1 - self.alpha) * self.value + self.alpha * median_bpm

        return round(self.value)

def init_i2c_sensors():

    ox = max30100.MAX30100(
        i2c_bus=1,
        mode=max30100.MODE_SPO2,
        sample_rate=max30100.SAMPLE_RATE_100,
        led_current_red=DEFAULT_LED_CURRENT,
        led_current_ir=DEFAULT_LED_CURRENT,
        pulse_width=max30100.PULSE_WIDTH_1600US_ADC_16,
        max_buffer_len=WINDOW_SIZE,
    )
    ox.enable_spo2()
    log.info("MAX30102 initialised at 0x57 on bus 1 (default)")

    from adafruit_extended_bus import ExtendedI2C
    i2c_bus2 = ExtendedI2C(MLX90614_I2C_BUS_NUMBER)
    mlx = adafruit_mlx90614.MLX90614(i2c_bus2)
    log.info("MLX90614 initialised at 0x5A on bus %d", MLX90614_I2C_BUS_NUMBER)

    return mlx, ox

def init_gps():

    gps_serial = serial.Serial(
        port=GPS_PORT,
        baudrate=GPS_BAUD,
        timeout=GPS_TIMEOUT,
    )
    log.info("GPS UART opened on %s @ %d baud", GPS_PORT, GPS_BAUD)
    return gps_serial

def main():
    print("=" * 64)
    print("  T-BTN Army Dog Wearable - MAX30102 + NEO-6M GPS + MLX90614")
    print("  Offline-first mode: data is buffered locally and uploaded")
    print("  when internet (hotspot / 4G dongle) is available.")
    print("=" * 64)

    try:
        mlx, ox = init_i2c_sensors()
        print("[OK] MAX30102 (bus 1) + MLX90614 (bus "
              f"{MLX90614_I2C_BUS_NUMBER}) initialised")
    except (RuntimeError, ValueError) as exc:
        print(f"[FAIL] I2C sensor init failed: {exc}")
        print("       For MAX30102: run `i2cdetect -y 1` and confirm 0x57 appears.")
        print(f"       For MLX90614 on bus {MLX90614_I2C_BUS_NUMBER}: do NOT rely on")
        print("       i2cdetect - it is documented as unreliable for this exact")
        print("       sensor on Pi 5 even when wiring is correct (see Raspberry Pi")
        print("       GitHub issue raspberrypi/bookworm-feedback
        print("       is happening, double check VIN/GND/SCL/SDA wiring and the two")
        print("       pull-up resistors on pins 15/16 instead.")
        raise SystemExit(1) from exc
    except Exception as exc:
        log.error("I2C initialisation failed: %s", exc)
        raise SystemExit(1) from exc

    try:
        gps_serial = init_gps()
        print("[OK] GPS UART opened on", GPS_PORT)
    except Exception as exc:
        log.warning("GPS UART failed to open (%s) - continuing without GPS", exc)
        print(f"[WARN] GPS UART failed to open ({exc}) - continuing without GPS")
        gps_serial = None

    bpm_smoother = BPMSmoother()

    body_temp = None
    bpm       = None
    spo2      = None
    gps_lat   = None
    gps_lon   = None
    gps_speed_kmh = None
    gps_heading   = None
    saturated = False

    t_last_print   = time.monotonic()
    t_last_mlx     = time.monotonic()
    t_last_publish = time.monotonic()
    t_last_dsp     = time.monotonic()

    try:
        from firebase_client import FirebaseClient
        firebase = FirebaseClient()
        _firebase_ref = firebase
    except Exception as exc:
        log.warning("Failed to load Firebase Client: %s. Running sensor-only mode.", exc)
        firebase = None

    gps_line_buf = ""

    print("\nPlace fingertip gently on the MAX30102 sensor for BPM/SpO2.")
    print("Sampling... (Ctrl-C to quit)\n")

    while True:
        loop_start = time.monotonic()

        try:
            ox.read_sensor()
        except OSError as exc:
            log.debug("MAX30102 I2C error (skipping sample): %s", exc)
        except Exception as exc:
            log.warning("MAX30102 unexpected error: %s", exc)

        now = time.monotonic()
        if now - t_last_mlx >= MLX_INTERVAL:
            t_last_mlx = now
            try:
                body_temp = round(mlx.object_temperature, 1)
            except OSError as exc:
                log.debug("MLX90614 I2C error: %s", exc)
            except Exception as exc:
                log.warning("MLX90614 unexpected error: %s", exc)

        if gps_serial is not None:
            try:
                available = gps_serial.in_waiting
                if available:
                    raw_bytes = gps_serial.read(available)
                    gps_line_buf += raw_bytes.decode("ascii", errors="replace")

                    if len(gps_line_buf) > 4096:
                        gps_line_buf = gps_line_buf[-4096:]

                    while "\n" in gps_line_buf:
                        sentence, gps_line_buf = gps_line_buf.split("\n", 1)
                        sentence = sentence.strip()
                        if not sentence:
                            continue
                        try:
                            msg = pynmea2.parse(sentence)

                            if hasattr(msg, "latitude") and hasattr(msg, "longitude"):
                                lat = msg.latitude
                                lon = msg.longitude
                                if lat != 0.0 or lon != 0.0:
                                    gps_lat = lat
                                    gps_lon = lon

                            if isinstance(msg, pynmea2.types.talker.RMC):
                                if msg.spd_over_grnd is not None:
                                    try:
                                        gps_speed_kmh = round(float(msg.spd_over_grnd) * 1.852, 1)
                                    except (ValueError, TypeError):
                                        pass
                                if msg.true_course is not None:
                                    try:
                                        gps_heading = round(float(msg.true_course), 1)
                                    except (ValueError, TypeError):
                                        pass
                        except pynmea2.ParseError:
                            pass
                        except Exception as exc:
                            log.debug("NMEA parse error: %s", exc)
            except serial.SerialException as exc:
                log.warning("GPS serial error: %s", exc)
            except Exception as exc:
                log.warning("GPS unexpected error: %s", exc)

        now = time.monotonic()
        if now - t_last_dsp >= DSP_INTERVAL:
            t_last_dsp = now
            try:
                ir_buf  = list(ox.buffer_ir)
                red_buf = list(ox.buffer_red)

                ir_mean_fast = np.mean(ir_buf) if len(ir_buf) > 0 else 0.0
                no_contact = (
                    len(ir_buf) > 0
                    and ir_mean_fast < NO_CONTACT_IR_THRESHOLD
                )

                if no_contact:

                    bpm  = None
                    spo2 = None
                    saturated = False
                    bpm_smoother._history.clear()
                    bpm_smoother.value = None
                    ox.buffer_ir.clear()
                    ox.buffer_red.clear()
                else:
                    raw_bpm, raw_spo2 = compute_bpm_spo2(ir_buf, red_buf, SAMPLE_RATE_HZ)
                    if raw_bpm == "SATURATED":
                        saturated = True
                        bpm = None
                    else:
                        saturated = False
                        bpm = bpm_smoother.update(raw_bpm)
                        if raw_spo2 is not None:
                            spo2 = raw_spo2
            except Exception as exc:
                log.warning("DSP error: %s", exc)

        now = time.monotonic()
        if now - t_last_print >= PRINT_INTERVAL:
            t_last_print = now

            temp_str = f"{body_temp:.1f}*C" if body_temp is not None else "---*C"

            if gps_lat is not None and gps_lon is not None:
                gps_str = f"{gps_lat:.4f}, {gps_lon:.4f}"
            else:
                gps_str = "No Fix"

            if saturated:
                ir_now  = ox.ir  if ox.ir  is not None else 0
                red_now = ox.red if ox.red is not None else 0
                print(
                    f"Temp: {temp_str} | "
                    f"BPM: SATURATED (raw IR:{ir_now} RED:{red_now}) | "
                    f"GPS: {gps_str}"
                )
            else:
                bpm_str  = str(bpm)  if bpm  is not None else "---"
                spo2_str = str(spo2) if spo2 is not None else "---"

                ir_mean = sum(list(ox.buffer_ir)) / len(ox.buffer_ir) if len(ox.buffer_ir) > 0 else 0
                print(
                    f"Temp: {temp_str} | "
                    f"BPM: {bpm_str} (SpO2: {spo2_str}%) | "
                    f"GPS: {gps_str} | IR: {ir_mean:.0f}"
                )

        if firebase is not None:
            if now - t_last_publish >= PUBLISH_INTERVAL:
                t_last_publish = now

                throttle_flags = -1
                try:
                    out = subprocess.check_output(
                        ["vcgencmd", "get_throttled"], text=True, timeout=1
                    )
                    throttle_flags = int(out.strip().split("=")[1], 16)
                except Exception:
                    pass

                data = {
                    "deviceId":      DEVICE_ID,
                    "temp":          body_temp,
                    "bpm":           bpm if not saturated else None,
                    "spo2":          spo2 if not saturated else None,
                    "lat":           gps_lat,
                    "lon":           gps_lon,
                    "speedKmh":      gps_speed_kmh,
                    "heading":       gps_heading,
                    "throttleFlags": throttle_flags,
                }
                firebase.publish_reading(data)

        elapsed = time.monotonic() - loop_start
        sleep_for = POLL_INTERVAL - elapsed
        if sleep_for > 0:
            time.sleep(sleep_for)

_firebase_ref = None

def _handle_sigterm(signum, frame):

    import sys
    print("\n[T-BTN] SIGTERM received - flushing buffer and shutting down...")
    if _firebase_ref is not None:
        _firebase_ref.stop()
    sys.exit(0)

if __name__ == "__main__":
    import signal
    signal.signal(signal.SIGTERM, _handle_sigterm)

    try:
        main()
    except KeyboardInterrupt:
        print("\n[T-BTN] Keyboard interrupt - flushing buffer and shutting down...")

            _firebase_ref.stop()
